# CRITICAL FIX: Wrong Stripe ID Type - Using Product ID Instead of Price ID

## Error Analysis

```
❌ Error creating checkout session: No such price: 'prod_TDF20nQ6n4TUeu'
```

## Root Cause

The code is sending **Product IDs** (starting with `prod_`) instead of **Price IDs** (starting with `price_`).

**Stripe requires:**
- ✅ Price IDs: `price_1234567890abcdef`
- ❌ Product IDs: `prod_1234567890abcdef` (WRONG)

## The Problem

Your environment variables OR frontend code is using Product IDs when it should use Price IDs.

---

## Fix #1: Get Correct Price IDs from Stripe Dashboard

### Step 1: Go to Stripe Dashboard
1. Log into https://dashboard.stripe.com
2. Navigate to **Products** in the left sidebar
3. Click on each of your products (Starter, Standard, Pro, Top-Up)

### Step 2: Copy the PRICE ID (not Product ID)
For each product, you'll see:
```
Product ID: prod_ABC123xyz (DON'T USE THIS)
```

## Fix #3: Validate Price IDs on Backend

### Location: `server/routes.ts`

Add validation before creating checkout session:

```typescript
app.post("/api/create-checkout-session", requireAuth, async (req, res) => {
  try {
    const { priceId, planType } = req.body;
    const userId = req.session.userId;
    
    // CRITICAL: Validate priceId format
    if (!priceId) {
      return res.status(400).json({ error: 'Price ID is required' });
    }
    
    if (!priceId.startsWith('price_')) {
      console.error(`❌ Invalid Price ID format: ${priceId}`);
      return res.status(400).json({ 
        error: `Invalid Price ID format. Received: ${priceId}. Must start with 'price_'` 
      });
    }
    
    console.log('✅ Creating checkout with valid Price ID:', priceId);
    
    // ... rest of checkout logic
    
    // Verify price exists in Stripe before creating session
    try {
      await stripe.prices.retrieve(priceId);
      console.log('✅ Price verified in Stripe');
    } catch (error) {
      console.error('❌ Price not found in Stripe:', priceId);
      return res.status(400).json({ 
        error: `Price ${priceId} not found in Stripe. Please check environment variables.` 
      });
    }
    
    // Create checkout session
    const session = await stripe.checkout.sessions.create({
      // ... checkout config with priceId
    });
    
    res.json({ url: session.url });
    
  } catch (error) {
    console.error('❌ Checkout error:', error);
    res.status(500).json({ 
      error: 'Failed to create checkout session',
      details: error.message 
    });
  }
});
```
